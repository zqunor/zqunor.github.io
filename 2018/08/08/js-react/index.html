（2）组件拆分

React 的组件开发意思是将页面上每个部分作为一个组件，然后通过每个组件之间的通信，进行数据交互，实现完整页面的渲染。

TodoList 功能包括用户输入部分和 List 清单部分，所以将整个功能拆分为两部分。用户输入部分为TodoList; List清单部分为TodoItem。
